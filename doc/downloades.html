Descarga

Obtenga la ultima version de Report Manager y sus paquetes.

Report Manager esta alojado en SourceForge. Descargue la ultima version desde la pagina del proyecto en SourceForge.

La version incluye los siguientes paquetes:

  • Disenador de informes Report Manager (Windows y Linux)
  • Paquetes de componentes para Delphi, C++Builder y FPC/Lazarus
  • Componente ActiveX de Report Manager
  • Biblioteca .NET y .NET Core

Los paquetes del disenador tambien incluyen el control ActiveX (Windows) y el servidor web de informes (Windows y Linux).

Para desarrollo en .NET hay paquetes disponibles en NuGet y en un repositorio de GitHub dedicado.
