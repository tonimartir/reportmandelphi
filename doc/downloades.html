Descarga

Obtenga la última versión de Report Manager y sus paquetes.

Report Manager está alojado en SourceForge. Descargue la última versión desde la página del proyecto en SourceForge.

La versión incluye los siguientes paquetes:

  • Diseñador de informes Report Manager (Windows y Linux)
  • Paquetes de componentes para Delphi, C++Builder y FPC/Lazarus
  • Componente ActiveX de Report Manager
  • Biblioteca .NET y .NET Core

Los paquetes del diseñador también incluyen el control ActiveX (Windows) y el servidor web de informes (Windows y Linux).

Para desarrollo en .NET hay paquetes disponibles en NuGet y en un repositorio de GitHub dedicado.
