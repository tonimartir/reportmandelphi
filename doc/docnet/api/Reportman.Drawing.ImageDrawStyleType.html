    Enum ImageDrawStyleType

    Drawing styles, determines the way the image adapt to a bounding box

    Namespace: Reportman.Drawing
    Assembly: Reportman.Drawing.dll
    Syntax
    public enum ImageDrawStyleType

    Fields

    Name Description
    Crop

    The image is drawn, but it's clipped inside the rectangle, part of the image can be not visible
    Full

    The image is drawn without clipping, so bounding box size is ignored, dots per inch is used to calculate size
    Stretch

    The image is adapted to the bounding box size
    Tile

    The image is drawn as many times as necessary to fill the rectangle left to right and top to bottom, scaling will depend on ouput device
    Tiledpi

    The image is drawn as many times as necessary to fill the rectangle left to right and top to bottom, the dots per inch image attribute will remain
