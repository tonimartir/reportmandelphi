Servidor Web de Reportes

El servidor web de Report Manager genera archivos PDF al instante, solicitando al usuario el nombre del informe y los parámetros, luego ejecuta el informe y devuelve un archivo en formato Adobe PDF. Si tiene un plugin puede visualizarlo directamente en el navegador.
