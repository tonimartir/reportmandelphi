Setup printreptopdf compilation development environment

To compile reportman printreptopdf, the LinuxPAServer crosscompiler is used. A connection to PAServer is needed. The new versions of printreptopdf can handle complex shaping fonts like arabic fonts, so some development libraries must be installed.
  • Installation of freetype devel. RHLE sudo dnf install freetype-devel
  • Installation of harfbuzz devel. RHLE sudo dnf install harfbuzz-devel
  • Installation of harfbuzz font subsetting if available. RHLE sudo dnf install libharfbuzz-subset0
  • Installation of libicu devel. RHLE sudo dnf install libicu-devel
  • Edit SDK and update to see changes on Linux development cache
