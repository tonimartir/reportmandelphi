Units documentation - rpmdshfolder.pas

Units documentation

rpmdshfolder.pas

Description

This unit implements obtaining local configuration file paths, in a way compatible with Windows (Windows 10/11 and Windows Server) and Linux environments.

Functions

Declaration Description
function Obtainininameuserconfig (company, product, filename:string):string; Home user configuration file
function Obtainininamelocalconfig (company, product, filename:string):string; Machine based configuration file, shared with all users
function Obtainininamecommonconfig (company, product, filename:string):string; System configuration usually you need Administrator or root privileges
