Report Manager - What is new

Release notes

What is new

Version 4.0.10 include this new features (June 2026)

  • AI Copilot in the Report Designer to help you edit, create, debug and analyze your reports.
  • AI report generation from an optional schema: provide a schema and the AI will generate a new report for you. Total privacy — only the schema is sent to the AI, never your data. Schemas are defined and bound on app.reportman.es and can be reused to build KPIs in the new Reportman app.
  • AI Expression assistant to help you create and refine report expressions.
  • New AI-enabled SQL autocomplete in the SQL editor.
  • New AI natural language query creation and modification — describe what you need in plain language and the AI writes or edits the SQL query for you.
  • New report connection wizard to set up your report data source, aligned with your saved connections (direct database or Reportman DB Agent).
  • Optional HTML rendering for labels and expressions: font name, size, style and color via HTML, with expression interpolation using {{EXPRESSION}}.
  • Reportman DB Agent connections — execute your reports from everywhere.
  • Undo and redo support in the Report Designer, with persistent history.
  • Docker-enabled Report Server — REST API and self-hosted web (repweb) for Linux/Apache.
  • .Net Report Designer published as beta — its first general public release.
  • More information at ai.reportman.es.

Version 3.9.15 include this new features December 2025

  • PDF output complex text shaping support (arabic,devanagari etc..)
  • Complex text shaping implements font fallback
  • To enable complex shaping set bidi mode to partial to text components
  • Rework font selection in Linux now using fontconfig if installed
  • PDF Font subsetting in Linux will use Harfbuzz if installed
  • Delphi: 13 component packages
  • Updated database drivers: OpenODBC, FireDac, DBExpress, Firebird, Zeos
  • .Net nuget packages published
  • Experimental .net standalone designer published
  • Updated Linux printreptopdf command line tool

Version 3.7.5 include this new features December 2024

  • PDF/A3 conformance at page setup
  • PDF Document metadata at page setup
  • Embedded files in report template, will be included in pdf output PDF/A3
  • Updated ActiveX, can add embedded files, set metadata and PDF Conformance
  • Delphi: 11.0, 12.X component packages
  • Updated database drivers: OpenODBC, FireDac, DBExpress, Firebird, Zeos
  • Breaking change: Report templates saved with this version can not be opened with older versions
  • Fixes: QR Barcode ActiveX and Delphi fixes
  • Updated .Net versions

Version 3.4.7 include this new features October 2021

  • QR Code Support
  • New Linux 64bit printreptopdf executable
  • Delphi: 10.3 and 10.4 component packages
  • .Net Core Libraries
  • .Net library: Qrcode barcode updated.
  • Font size and autoscale configuration for charts
  • .Net Report designer (beta)
  • Updated database drivers: OpenODBC, FireDac, DBExpress, Firebird, Zeos
  • Breaking change: Report templates saved with this version can not be opened with older versions

Version 3.1 include this new features (February 2016)

  • New 64bit binaries
  • All the database drivers and resources are statically linked into the binaries, this means larger executable files, but without dependencies
  • Enhanced support for .net runtime report execution from the designer
  • Designer support for Windows 10 and other themes at preferences menu
  • Enhanced Unicode support
  • Compatibility fixes for web server in Windows Server 2008 and later.
  • Enhanced compatibility for Windows 8, Windows 10 and high dpi screens.
  • Database drivers updated (DbExpress and Zeos).
  • New FireDac Database Drivers
  • .Net library updated, added Qrcode barcode and charting, bug fixes.
  • .Net support for sqlite and mysql
  • Alpha preview version of .Net Designer included in net2 folder.

Version 2.9 include this new features (January 2010)

  • Unicode support for some database drivers (DAO included, if the driver supports it)
  • Enhanced compatibility with Windows 7 and Vista, some video drivers worked incorrectly when Themes were enabled.

  • .Net library updated, dot matrix printer support and bug fixes.
  • Enhanced unicode support for PDF output.

Version 2.8 include this new features (June 2009)

  • Updated zeos database drivers (Windows-Linux)
  • Windows Vista compatibility (some video drivers caused problems before)
  • Updated database drivers (Windows).
  • Enhanced Dot net port. (Excel output implemented,Preview is now working in Linux because new Mono releases).
  • Added compatibility for Delphi 2007/C++Builder 2007 and Delphi 2009
  • Added compatibility for Indy version 10.
  • Bugfixes and small enhancements in the Designer.

  • New parameter type, multiple selection type parameter.
  • Search dataset for parameters, allowing the user to search for parameter values

  • Experimental Unicode fonts truetype support on PDF export.
  • Report file can be stored as XML (preferred save format), experimental.
  • Bugfixes

Version 2.3 include this new features (January 2005)

  • Great performance enhancements in Linux, linked querys, subreport and others.
  • Startup performance enhancement for all command line tools.
  • Truetype font embedding speed up
  • Enhanced ActiveX control, printer selection, saving metafiles...
  • Other minor enhancements (default font in designer,optimized tiled draw, obj.inspector font selection...)
  • Partial unicode enabled designer (object inspector) for Windows NT systems, Linux already supports this.
  • Group page count element implementation.
  • PDF417 barcode printing (2d barcode allowing 1024 bytes per barcode)
  • Enhanced export to Excel (single sheet-multiple sheet)
  • Use environment variable (REPMANUSEKPRINTER=true) to integrate the print engine with KDE.
  • Remote report parameter ask and modify, from components, ActiveX and shared library API.
  • Allow different page sizes, orientations and paper bins, in a single document (Windows,PDF, experimental)
  • Report language selection at runtime (M.LANGUAGE for example to print several invoices in different languages in the same report)
  • New expression evaluator functions (TextHeight, ...)
  • Delphi 2005 support
  • Since 2.2 version, for client/server report communications, the client application must be at least same version as server, older versions require same version in client and server to work.
  • Bugfixes

Version 2.0 include this new features (December 22th 2004)

  • Native .Net support, compiled with Delphi for .Net.
  • HTML output
  • Full Report Library support (reports stored inside database tables with a tree structure)
  • Enhanced security model for web server and TCP report server
  • Enhanced ActiveX

  • Report compiler (Windows only), generate a executable independent report (About 300Kbytes).
  • Enhanced Borland C++ Builder 6 support.

Version 1.6 include this new features (August 7th 2003)

  • Multiprocessor support (smp).
  • New output driver, text only with printer escape codes to print lighting fast in any dot matrix printer, in Windows and Linux, also to save the report result as plain text.
  • New shared libraries libreportmanapi.so and libreportmanapiqt.so to access the engine with GNU C and any language capable of calling dynamic link library functions (Reportman.ocx for Windows)
  • Zeos library drivers support
  • Excel export support for Microsoft Windows
  • Allow report definition streaming in three formats binary-zlib, binary and text , so you can track changes in your reports with cvs
  • Plain text file input support as a database file with visual configuration dialog
  • New method for VCL components and ActiveX, executeremote to execute reports stored in report servers, also available in ActiveX and GNU C interface.
  • New property editors for Delphi 5 (TRpAlias)
  • Enhanced command line tools (-preview -pdialog options added), all tools can work now with standard input and standard ouput.
  • ActiveX composite reports feature.
  • New methods for drawing text and graphic elements usable from the expression evaluator.
  • Enhanced designer - New database connections/datasets configuration dialog, active help in properties in the object inspector...
  • Enhanced support for Borland C++ Builder (added CLX package and more documentation)
  • Enhanced documentation
  • Portuguese language support thanks to Warlei Alves
  • Bug fixes
  • Report Manager is now a Project Jedi Alliance member
  • Notes: This upgrade requires updates for client and server, that is a 1.5 client does not work with a 1.6 server and a 1.6 client does not work with 1.5 server. This is also true for ActiveX Internet explorer plugin. Replace dbxdrivers old file with the new one for zeos support.
