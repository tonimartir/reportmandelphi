Servidor de informes web

El servidor web de Report Manager genera archivos PDF al vuelo, preguntando al usuario los parámetros del informe, ejecuta el informe y devuelve un archivo con formato Adobe PDF. Si su navegador dispone del plugin adecuado, podrá visualizar el archivo dentro de la ventana del navegador.
