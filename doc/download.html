Download

Get the latest Report Manager release and packages.

Report Manager is hosted on SourceForge. Download the latest release from the SourceForge project page.

The release includes the following packages:

  • Report Manager Designer application (Windows and Linux)
  • Component packages for Delphi, C++Builder and FPC/Lazarus
  • Report Manager ActiveX component
  • .NET and .NET Core library

The designer packages also include the ActiveX control (Windows) and the Web Report Server (Windows and Linux).

For .NET development, packages are available on NuGet and in a dedicated GitHub repository.
